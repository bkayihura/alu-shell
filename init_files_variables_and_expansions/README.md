# Shell Variables and Expansions

0. 0-alias: Creates an alias ls=rm *
1. 1-hello_you: Prints hello user
2. 2-path: Adds /action to PATH
3. 3-paths: Counts directories in PATH
4. 4-global_variables: Lists environment variables
5. 5-local_variables: Lists all variables and functions
6. 6-create_local_variable: Creates local variable BEST=School
7. 7-create_global_variable: Creates global variable BEST=School
8. 8-true_knowledge: Adds 128 to TRUEKNOWLEDGE
9. 9-divide_and_rule: Divides POWER by DIVIDE
10. 10-love_exponent_breath: BREATH to the power of LOVE
11. 11-binary_to_decimal: Converts binary to decimal
12. 12-combinations: Prints letter combinations except oo
13. 13-print_float: Prints number with 2 decimal places
14. 14-decimal_to_hexadecimal: Converts decimal to hex
15. 15-rot13: Encodes/decodes text using rot13
16. 16-odd: Prints every other line
17. 17-water_and_stir: Adds numbers in custom bases
